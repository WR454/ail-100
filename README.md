# منصة الخبير التقني الشامل (Tech-Expert Hub)

## الرؤية العامة
تطوير منصة ذكية متكاملة تعمل كمساعد تقني فوري مدعوم بالذكاء الاصطناعي. تهدف إلى تقديم حلول تقنية تخصصية دقيقة عبر مكتبات معرفية مصنفة، مع التركيز على الاحترافية في التصميم والسرعة الفائقة في الرد.

## المكتبات التقنية المتخصصة
- الأمن السيبراني
- نظم المعلومات
- تقنية المعلومات
- تحليل المعلومات
- هندسة البرمجيات

## التقنيات المستخدمة
- Backend: Python + FastAPI
- Frontend: React + Vite
- AI: OpenAI GPT-4o

## كيفية التشغيل
1. Backend: `python -m uvicorn backend.main:app --reload`
2. Frontend: في مجلد frontend، `npm install` ثم `npm run dev`

## API
- GET / : رسالة ترحيب
- GET /modules : قائمة المكتبات
- POST /ask : سؤال مع module و question
